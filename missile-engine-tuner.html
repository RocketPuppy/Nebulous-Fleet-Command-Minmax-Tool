Missile Engine Tuner

Use this tool to precisely tune your missile engine performance. First, upload a missile template. The tool will show you the current performance values for your missile engine. You can then adjust the values by moving the sliders or entering numbers directly. You can hold a value constant so you can adjust the other two values without changing the held value.

Press Download to download the missile file. You can overwrite your old template with it (make a backup if you are concerned about losing it).

Some values might be slightly off from their in game values.

This tool is provided as-is. It may generate corrupted missile templates. If you want to keep your templates safe, make a backup before replacing them.
