Nebulous: Fleet Command · Min-Max Tool

Welcome to the Nebulous: Fleet Command min-max tool! You can use this tool to optimize your ships, missiles, and strategies to level up your game! Select from one of the tools below to get started!

  • Missiles vs. PD: This tool lets you compare different missile builds with different point defense setups.
  • Damage Mechanics: This tool lets you explore the different damage mechanics against various targets.
  • Movement Mechanics: This tool lets you explore the different movement mechanics in various scenarios.
  • Missile Engine Tuner: This tool let's you precisely tune your missile engine performance.
  • Skirmish Report Viewer: This tool let's you view your after action skirmish reports.
  • Skirmish Report Stats: This let's you view stats from several skirmish reports.
  • Ship Builder: This let's you build ship templates to download into an existing fleet.
  • Test Branch Tools: Tools for making things easier on the public test branch.
